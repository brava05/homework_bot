Это бот, который проверяет наличие новых данных про проверенные домашние задания
на API яндекс практикума и отправки изменений в Telegram.

Написано на python 3.


Как использовать.
Сначала надо установить python.

Потом клонируем репозиторий себе на компьютер:

$ git clone git@github.com/USERNAME/{{ project_name }}.git

Переходим в папку проекта
$ cd {{ project_name }}

Создаем и активируем виртуальное пространство

python -m venv venv
source venv/Scripts/activate

Устанавливаем все необходимые пакеты:
$ pip install -r requirements/local.txt

Автор: Бражинский Валерий.
